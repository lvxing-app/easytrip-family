# 一路简行 · 可操作旅行 App 演示版

这是一个无需安装依赖即可体验的手机端 Web App 演示，覆盖国内游和出境游。

项目已适配 GitHub Pages 等静态网站托管，发布时保留 `.nojekyll` 文件即可。

当前公开网址为 `https://lvxing-app.github.io/easytrip-family/`，第一版不依赖中国大陆域名或自定义域名。全球优先部署、地区适配规则和后续路线图见 `docs/global-first-plan.md`。

## 如何体验

直接打开 `index.html` 即可操作。若通过本地网页服务器打开，还可以安装为 PWA，并使用离线缓存。

建议依次体验：

1. 在“国内游 / 出境游”之间切换。
2. 修改自然语言需求、日期、同行人、预算和节奏。
3. 生成完整行程，更换景点并重新优化路线。
4. 在“预订服务”中体验交通、酒店、门票和当地服务。
5. 进入“旅途中”体验导航、翻译、家庭共享、订单和紧急求助。
6. 右上角打开“大字简洁模式”和“连接中心”。
7. 点击顶部语言按钮，在12种界面语言之间切换；阿拉伯语会自动启用从右向左布局。
8. 海外目的地可直接选择东京、首尔、曼谷、巴黎、伦敦、纽约和新加坡；自然语言输入同时识别常用英文与当地名称。
9. 点击“保存离线旅行包”，把当前行程、订单选择和目的地紧急信息保存在本机。

## 多语言

当前提供简体中文、英语、日语、韩语、法语、西班牙语、德语、意大利语、泰语、阿拉伯语、俄语和葡萄牙语。语言选择会保存在浏览器中，并同步影响导航、行程标题、日期、提醒、预订状态和旅途中助手。

景点专有名称默认保留官方名称，行程说明通过统一翻译层展示。正式上线时可将该翻译层替换为真实机器翻译和人工校对内容。

## 家庭群聊

底部“家庭”、行程页顶部“分享”和旅途中“家庭群聊”都可以进入家庭空间。当前提供两种运行模式：

- 云端模式：全球邮箱验证码、`+86` 手机短信验证码、跨设备家庭与实时聊天
- 本机演示：云环境尚未配置时自动启用，验证码固定为 `123456`
- 使用手机号或邮箱形式注册、退出和重新登录账号
- 创建家庭，或使用家庭邀请码加入家庭
- 像家庭群聊一样发送文字消息；云端模式会实时同步到其他设备
- 将当前完整旅行计划作为行程卡片发送到群聊
- 从群聊行程卡片返回查看完整行程
- 复制家庭邀请码和退出家庭

真实家庭后台的前端适配器、云函数、数据库安全规则和测试版规格已经加入仓库。公开网站在填入 CloudBase 环境 ID、Publishable Key 并部署 `family-service` 前仍保持本机演示模式。完整启用步骤见 `docs/family-pilot-spec.md`。

## 地图

行程页会根据目的地自动选择地图服务：

- 中国大陆行程：高德地图 AMap
- 海外行程：Google Maps
- 尚未配置正式密钥时：使用 OpenStreetMap 路线预览，同时保留高德或 Google Maps 外部导航

在 `map-config.js` 中填写地图凭证即可启用正式内嵌地图。高德需要 Web JS API Key，并按官方建议在服务端配置安全密钥代理；Google Maps JavaScript API 需要启用项目、结算账号和受来源限制的 API Key。

演示使用 OpenStreetMap 时只加载用户当前查看范围，不提供地图瓦片批量下载或离线预取。

## 目的地地区适配

行程会按目的地自动切换当地货币、时区、地图提供商、语言提示和官方紧急号码。当前已配置中国大陆、东京、首尔、曼谷、巴黎、伦敦、纽约和新加坡。界面语言仍由用户选择，不会因目的地变化而强制切换。

价格显示为当地货币演示估算，不代表实时库存或成交价；正式上线后需由后端接入实时汇率与供应商报价。

## 当前连接方式

演示版的所有服务都通过 `app.js` 中的 `IntegrationHub` 统一调用，目前使用本地模拟数据，不会创建真实订单或产生费用。

“连接中心”会显示每项服务的实际状态：地图正式密钥、路线预览、家庭云端或本机模式、离线旅行包、多语言以及真实预订准备情况。未配置的服务会明确显示“需要配置”或“仅演示”。

正式上线时，需要在后端配置并替换以下服务：

- 国内及海外地图与路线服务
- 酒店库存、价格和订单服务
- 12306、航司或 GDS 交通票务服务
- 国内及全球景点票务服务
- 国内及全球天气服务
- 多语言翻译和语音服务
- 实时汇率、多币种支付和退款服务
- 旅行保险、紧急救助和领事信息服务

API 密钥、支付签名和商务凭证必须保存在服务端，不能直接写在前端文件中。

## 文件结构

- `index.html`：页面结构与可访问性标记
- `styles.css`：手机端界面、响应式布局及大字模式
- `app.js`：交互逻辑、旅行数据和统一服务接口
- `family-chat.js`：家庭账号、家庭房间、聊天和行程卡片共享
- `i18n.js`：多语言资源、日期格式和从右向左布局配置
- `map-config.js`：高德与 Google Maps 配置入口
- `cloud-config.js`：家庭云端环境开关与公开配置入口
- `family-cloud.js`：CloudBase 登录、云函数调用和实时消息订阅适配器
- `cloudfunctions/family-service`：家庭、成员、聊天和共享行程的受保护云函数
- `cloudbase/database-rules`：家庭数据集合的客户端安全规则
- `docs/family-pilot-spec.md`：20–50 个家庭首批测试版规格与验收标准
- `tests/smoke.cjs`：手机端界面、账号、家庭、聊天、共享行程和重新登录回归测试
- `tests/family-service.cjs`：家庭云函数内存集成测试
- `tests/cloudbase-sdk-contract.cjs`：CloudBase 浏览器 SDK 在线接口契约检查
- `maps.js`：地图提供商切换、地点标记、路线预览和外部导航
- `manifest.webmanifest`：可安装 App 配置
- `service-worker.js`：离线缓存
- `assets/app-icon.svg`：App 图标
